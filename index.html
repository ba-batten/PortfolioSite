Frogger Clone

Frogger Clone demonstrates used of object oriented programming using Javascript. The game engine was provided by Udacity to comlete this project as part of the Front-End Web Developer Nanodegree. My task was to create the individual objects that make up the game such as the main character, enemy bugs, and collectables. The main player tries to collect as many gems as possible while avoiding the bugs moving across the screen. The game tracks the current score and the high score for the current session.
